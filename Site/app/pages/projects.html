Arcusical

Snapshot of repo on Github

Windows 8 music player and ongoing project over the last year. Written in C++ and C++/CX. Highlights of the project include an MPEG4 parser I wrote and an ALAC decoder that is integrated with Microsoft Media Foundation. The goals of the application are to be a music player that can handle ALAC, AAC, MP3, FLAC and WAV audio. Additionally it must also effectively handle duplicate songs (because my library has many) and present the albums very nicely. Nice to haves would be have audio transcoding features so my phone can easily sync my library without winding up with a bunch of lossless audio filing up the limited disk space.

This has been an ongoing project that I am building both for my own benifit (to address what I feel are features lacking in many existing player) and for enjoyment as I really enjoy working on this project.

Autonomy

Autonomy on Github

Experiemental C# project to test an idea I had for a parallelization algorithm. Autonomy is a Windows 8 Application that simulates several independant entities moving around a 2D space that update their position and collide on multiple threads but have no internal syncronization and nearly no locks. Uses Direct2D via SharpDX to draw the UI. For more details see the readme attached to the Github project.
